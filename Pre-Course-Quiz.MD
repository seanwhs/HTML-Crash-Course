# **HTML Pre-Course Quiz**

**Instructions:** Answer the following questions to the best of your ability. This quiz is to assess your prior knowledge.

---

### **Section 1: Multiple Choice**

1. Which of the following is a markup language used to create webpages?
   a) HTML
   b) CSS
   c) Python
   d) Java

2. What does a web browser do?
   a) Stores files on your computer
   b) Displays web content using HTML, CSS, and JavaScript
   c) Creates a programming language
   d) Edits videos

3. Which of the following is used to make a webpage look good visually?
   a) HTML
   b) CSS
   c) JavaScript
   d) SQL

4. Which of these file extensions is commonly used for HTML files?
   a) `.html`
   b) `.css`
   c) `.js`
   d) `.exe`

5. Which tag is used to create a heading in HTML?
   a) `<p>`
   b) `<h1>`
   c) `<div>`
   d) `<span>`

---

### **Section 2: True / False**

6. HTML is used to add style and color to a webpage. (True / False)
7. HTML uses tags like `<p>` and `<a>` to define content. (True / False)
8. You need special software to view a webpage in a browser. (True / False)
9. All webpages must have the `<html>` and `<body>` tags. (True / False)
10. A hyperlink is a clickable link that takes you to another page or website. (True / False)

---

### **Section 3: Short Answer / Opinion**

11. Have you ever written any code before? If yes, which language(s)?

12. What do you hope to learn from this HTML course?

13. Do you know what a “webpage structure” means? If yes, briefly describe it.

14. Have you ever created a table, list, or form on a webpage? (Yes/No)

15. Name one website or web application you use frequently and explain why you like its design.

---

### **Purpose / How to Use**

* **Multiple Choice & True/False:** Assess basic web literacy.
* **Short Answer / Opinion:** Understand learners’ prior experience and expectations.
* **Trainer Use:**

  * Group learners by experience level if needed.
  * Tailor examples and pace based on responses.
  * Identify common misconceptions to address during the course.
