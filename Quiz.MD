# **HTML 1-Day Crash Course Quiz**

### **Section 1: Multiple Choice**

1. What does HTML stand for?
   a) HyperText Markup Language
   b) HighText Machine Language
   c) Hyper Transfer Markdown Language
   d) Hyperlink Text Module Language

2. Which tag is used to create a paragraph?
   a) `<h1>`
   b) `<p>`
   c) `<div>`
   d) `<section>`

3. How do you make a link open in a new tab?
   a) `<a href="url">Link</a>`
   b) `<a href="url" target="_blank">Link</a>`
   c) `<a src="url">Link</a>`
   d) `<link href="url" newtab>`

4. Which tag is used to display an image?
   a) `<img>`
   b) `<picture>`
   c) `<image>`
   d) `<src>`

5. Which of the following is a semantic HTML tag?
   a) `<div>`
   b) `<span>`
   c) `<header>`
   d) `<b>`

### **Section 2: True / False**

6. `<h1>` is the largest heading tag in HTML. (True / False)
7. You can use `<ul>` and `<ol>` to create lists. (True / False)
8. The `<form>` tag is used to create tables. (True / False)
9. The `alt` attribute is used for accessibility and SEO in images. (True / False)
10. `<footer>` is used to add the bottom section of a webpage. (True / False)

### **Section 3: Short Answer**

11. Write the basic HTML boilerplate structure.

12. Create a simple HTML snippet for a link to `https://www.example.com` with the text “Visit Example”.

13. Write an HTML snippet for a table with 2 columns (Name, Age) and 2 rows of sample data.

14. Create a form snippet with inputs for Name (text), Email (email), and a Submit button.

15. Explain in one sentence why semantic HTML is important.

---

### **Answer Key**

**Multiple Choice:**

1. a
2. b
3. b
4. a
5. c

**True / False:**
6. True
7. True
8. False
9. True
10. True

**Short Answer:**

11.

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
  </body>
</html>
```

12.

```html
<a href="https://www.example.com" target="_blank">Visit Example</a>
```

13.

```html
<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>30</td>
  </tr>
</table>
```

14.

```html
<form>
  <label>Name: <input type="text" name="name"></label><br>
  <label>Email: <input type="email" name="email"></label><br>
  <input type="submit" value="Submit">
</form>
```

15. **Answer example:** Semantic HTML gives meaning to content, improves accessibility, and helps search engines understand the page.
