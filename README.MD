## **HTML Crash Course – 1 Day**

**Duration:** ~6–8 hours (can be adjusted)
**Goal:** Learn core HTML concepts and build a small webpage by the end of the day.

---

### **Morning Session (3–4 hours)**

**1. Introduction & Setup (30 min)**

* What is HTML and its role in web development
* Setting up a code editor (VS Code, Sublime Text, or similar)
* Creating your first HTML file

**2. Basic HTML Structure (30 min)**

* `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`
* Title, meta tags, and comments

**3. Text & Links (1 hour)**

* Headings (`<h1>`–`<h6>`), paragraphs (`<p>`), and text formatting (`<strong>`, `<em>`, `<mark>`)
* Lists (`<ul>`, `<ol>`, `<li>`)
* Hyperlinks (`<a>`) – internal & external links

**4. Hands-on Practice (1 hour)**

* Create a simple personal profile page with headings, paragraphs, and links

---

### **Afternoon Session (3–4 hours)**

**5. Images, Multimedia & Tables (1 hour)**

* Images: `<img>` with `src` and `alt`
* Audio and video embedding
* Simple tables: `<table>`, `<tr>`, `<td>`, `<th>`

**6. Forms (1 hour)**

* `<form>`, input types (`text`, `email`, `password`, `checkbox`, `radio`, `submit`)
* `<label>`, `<textarea>`, `<select>`/`<option>`
* Basic form validation

**7. Semantic HTML & Best Practices (30 min)**

* Semantic tags: `<header>`, `<footer>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`
* Accessibility basics (alt text, labels)

**8. Final Hands-on Project (1–1.5 hours)**

* Build a small multi-section webpage using:

  * Headings, paragraphs, images
  * Links and lists
  * A simple form
  * Semantic layout (header, footer, nav, main)

---

**Outcome:**
By the end of the day, you’ll have a fully structured HTML page and a solid understanding of the core HTML elements used in web development.
