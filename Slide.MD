### **Slide 2: What is HTML?**

* Stands for **HyperText Markup Language**
* Defines the **structure of webpages**
* Works with **CSS (style)** and **JavaScript (behavior)**

---

### **Slide 3: Setting Up**

* Install a code editor: VS Code, Sublime, Atom
* Create a new HTML file: `index.html`
* Structure:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
  </body>
</html>
```

---

### **Slide 4: Text & Headings**

* Headings: `<h1>` to `<h6>`
* Paragraphs: `<p>`
* Formatting: `<strong>`, `<em>`, `<mark>`

---

### **Slide 5: Lists & Links**

* Lists: `<ul>`, `<ol>`, `<li>`
* Links: `<a href="URL">Link Text</a>`
* Internal vs. external links

---

### **Slide 6: Images & Multimedia**

* Images: `<img src="image.jpg" alt="description">`
* Audio: `<audio src="audio.mp3" controls>`
* Video: `<video src="video.mp4" controls>`

---

### **Slide 7: Tables**

* Table structure:

```html
<table>
  <tr>
    <th>Header</th>
    <th>Header</th>
  </tr>
  <tr>
    <td>Data</td>
    <td>Data</td>
  </tr>
</table>
```

---

### **Slide 8: Forms**

* `<form>` with inputs:

  * Text: `<input type="text">`
  * Email: `<input type="email">`
  * Submit: `<input type="submit">`
* `<label>` and `<textarea>`
* Dropdowns: `<select>` / `<option>`

---

### **Slide 9: Semantic HTML**

* `<header>`, `<footer>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`
* Improves accessibility & SEO

---

### **Slide 10: Hands-On Project**

* Build a multi-section webpage:

  * Headings & paragraphs
  * Links & lists
  * Images & form
  * Semantic layout: header, footer, nav, main

---

### **Slide 11: Best Practices**

* Use semantic tags
* Write meaningful alt text
* Keep code clean & indented
* Test in multiple browsers
